GitToolBox
==========
Plugin for IntelliJ Idea expanding build-in Git integration.

[Plugin repository page](https://plugins.jetbrains.com/plugin/7499)

## Performance logging
Set following in **Help > Debug Log Settings...**
```
#zielu.gittoolbox.perf:trace
```
